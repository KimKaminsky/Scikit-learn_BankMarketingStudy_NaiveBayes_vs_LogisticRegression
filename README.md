# Scikit-learn_BankMarketingStudy_NaiveBayes_vs_LogisticRegression
Using scikit-learn to compare performance of Naive Bayes and Logistic Regression on Bank Marketing Study and to provide a recommendations about clients segments.

The purpose of this study is to advise a Portuguese bank interested in identifying factors that affect client responses to new term deposit offerings. This will help guide future telephone marketing campaigns. Two modelling techniques, Logistic Regression and Na√Øve Bayes, will be compared to determine which best segments the clients.
